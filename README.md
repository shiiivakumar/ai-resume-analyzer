## 🎯 What It Does
AI Resume Analyzer Pro helps job seekers beat the ATS filter by analyzing their resume against a target job role using Google Antigravity AI — delivering instant, actionable feedback.

## ✨ Features
- 📊 ATS Compatibility Score (0–100)
- 🔍 Keyword Gap Analysis — exact missing keywords
- ✅ Strengths Identification — what's already working
- ⚠️  Weakness Flagging — what's hurting your score
- 💡 Actionable Recommendations — 3–5 specific fixes
- ⚡ Instant Results — no login required
